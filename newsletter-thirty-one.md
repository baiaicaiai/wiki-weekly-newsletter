## 精品课程

![](images/java-Linux.jpg)

[《Java 程序员眼中的 Linux》](http://wiki.jikexueyuan.com/project/linux-in-eye-of-java/)——本文内容主要讲解 Linux 多种环境搭建，涉及 Java 开发的各个领域，是目前最全面的全环境安装课程。适合所有开发者，作者写作的目的也是为了总结，并与他人分享，希望得到一个系统的升华。

![](images/android0.jpg)

[《Android 百战经典》中文版](http://wiki.jikexueyuan.com/project/android-war/)——写下本系列文章的初衷是为了帮助更多 Android 的初学者可以尽快的进入角色，通过编写一些简单可行的代码，可以大大增加自己的兴趣和信心，同时也会加深对Android的学习和理解。

![](images/qt.jpg)

[《Qt 学习之路》](http://wiki.jikexueyuan.com/project/learn-road-qt/)——Qt 是一个著名的 C++ 库——或许并不能说这只是一个 GUI 库，因为 Qt 十分庞大，并不仅仅是 GUI。在本系列文章中，我们将使用 Qt4 进行 C++ GUI 的开发。

![](images/m-l.jpg)

[《神经网络与深度学习》](http://wiki.jikexueyuan.com/project/neural-networks-and-deep-learning-zh-cn/)——神经网络和深度学习是一本免费的在线书。本书会教会你：神经网络，一种美妙的受生物学启发的编程范式，可以让计算机从观测数据中进行学习，深度学习，一个强有力的用于神经网络学习的众多技术的集合。神经网络和深度学习目前给出了在图像识别、语音识别和自然语言处理领域中很多问题的最好解决方案。本书将会教你在神经网络和深度学习背后的众多核心概念。

![](images/android_java.jpg)

[《Google Java编程风格规范》](http://wiki.jikexueyuan.com/project/google-java-styleguide-zh/)——Google Java 风格指南，好的代码规范能够让团队更高效。本书适合有一定基础的开发人员。
